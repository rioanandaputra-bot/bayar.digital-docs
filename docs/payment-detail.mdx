# Payment Detail

Mengambil detail spesifik dari sebuah *invoice* berdasarkan `payment_code` yang terdaftar di sistem Anda.

Untuk melihat daftar keseluruhan *invoice*, silakan merujuk pada halaman [Payment List](./payment-list).

Gunakan *endpoint* ini untuk memeriksa status *invoice* kapan saja. Untuk mendapatkan pembaruan status secara *real-time*, kami sangat menyarankan Anda untuk mengatur [Webhook](./webhook).

Silakan lihat halaman [Status & Error Code](./status-code) untuk melihat daftar lengkap status pembayaran.
