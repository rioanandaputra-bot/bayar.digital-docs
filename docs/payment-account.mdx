# Payment Account

Endpoint ini digunakan untuk mengambil daftar rekening Anda yang tersedia sebagai tujuan pembayaran. 

Sebelum membuat invoice, Anda perlu memastikan ketersediaan rekening tujuan. Gunakan nilai `account_id` dari response endpoint ini sebagai parameter saat melakukan [Payment Create](./payment-create). 

**Penting:**
* Anda hanya dapat membuat pembayaran jika status `account_active = true` dan `account_quota > 0`.
* **Sistem Kuota Hari:** Anda dapat melakukan *Top Up / Renew* Kuota Hari melalui Dashboard. Kuota hanya akan berkurang **1 (satu)** apabila akun digunakan untuk membuat pembayaran pada hari tersebut, terlepas dari seberapa banyak request atau nominal pembayarannya. Jika tidak digunakan sama sekali pada hari itu, kuota Anda tidak akan berkurang.
