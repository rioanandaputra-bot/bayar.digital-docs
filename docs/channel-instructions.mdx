# Channel Instructions

Mengambil panduan atau langkah-langkah pembayaran untuk *channel* bank tertentu. Informasi ini dapat Anda tampilkan secara langsung pada antarmuka aplikasi Anda kepada pelanggan.

Setiap *channel* bank mungkin memiliki instruksi yang berbeda-beda. Gunakan `channel_id` yang didapatkan dari *response* [Payment Account](./payment-account) atau [Payment Create](./payment-create) untuk mengambil panduan langkah demi langkah tersebut.

## Response

<Tabs>
  <TabItem value="sukses" label="Sukses" default>
    <table>
      <thead>
        <tr><th>Field</th><th>Tipe</th><th>Deskripsi</th></tr>
      </thead>

        <tr><td>`channel_id`</td><td>uuid</td><td>ID *channel* terkait</td></tr>
        <tr><td>`instructions`</td><td>array</td><td>Daftar urutan langkah-langkah pembayaran</td></tr>

    **Format Objek di dalam `instructions`**

    <table>

        <tr><th>Atribut</th><th>Tipe</th><th>Deskripsi</th></tr>

      <tbody>
        <tr><td>`step`</td><td>int</td><td>Nomor urutan langkah</td></tr>

        <tr><td>`content`</td><td>string</td><td>Penjelasan detail dari langkah tersebut</td></tr>
      </tbody>
    </table>
