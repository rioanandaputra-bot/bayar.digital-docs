# Payment List

Mengambil daftar seluruh *invoice*, yang diurutkan mulai dari data yang paling baru dibuat.

Untuk melihat detail spesifik dari sebuah *invoice* tunggal, silakan kunjungi halaman [Payment Detail](./payment-detail).

Gunakan *endpoint* ini untuk keperluan rekonsiliasi data secara massal atau untuk menampilkan riwayat transaksi di dasbor sistem Anda. 

Silakan merujuk pada halaman [Status & Error Code](./status-code) untuk melihat daftar lengkap status pembayaran yang tersedia.
