# Payment List

Mendapatkan daftar invoice, diurutkan dari terbaru.

Untuk detail satu invoice, lihat halaman [Payment Detail](./payment-detail).

Gunakan endpoint ini untuk rekonsiliasi atau menampilkan history transaksi.

Lihat [Status & Error Code](./status-code) untuk daftar lengkap status payment.
